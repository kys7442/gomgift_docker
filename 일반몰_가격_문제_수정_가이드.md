# 일반몰 가격 문제 수정 가이드

## 개요

이 문서는 일반몰 전용 가격 관련 문제점들을 수정하기 위한 단계별 실행 가이드입니다.

## 일반몰 전용 문제점 요약

1. **AOV 상품 가격 저장 오류**: `ct_price`에 `it_price`(매입가) 저장 → 실제 판매가격은 `it_aov_price`에 저장되어야 함
2. **AOV 수량별 가격 미반영**: 수량이 많아져도 가격이 변하지 않음 (첫 번째 구간 값만 사용)
3. **JavaScript 가격 계산 오류**: `it_price` 사용 → 일반몰에서는 `it_price2` 사용해야 함

---

## 수정 작업 단계별 계획

### Phase 1: 일반몰용 AOV 가격 계산 함수 생성 (기반 작업)

#### 목적
- 일반몰에서도 수량별 AOV 가격을 정확히 계산할 수 있는 함수 생성
- 폐쇄몰의 `ClosedMallPriceService`와 유사하지만 일반몰용으로 단순화

#### 작업 내용

**1-1. 일반몰용 AOV 계산 함수 생성**

파일: `lib/aov_price_calculator_general.php` (신규 생성)

```php
<?php
if (!defined('_GNUBOARD_')) exit;

/**
 * 일반몰용 AOV 가격 계산 함수
 * 
 * 폐쇄몰의 ClosedMallPriceService와 유사하지만 일반몰용으로 단순화
 * - 할인률, 올림 단위 등 폐쇄몰 정책은 적용하지 않음
 * - 수량별 구간 가격만 계산
 * 
 * @param string $it_id 상품 ID
 * @param string $selected_option 선택한 옵션 값 (BASE/ADD의 col1 값)
 * @param string $row_type 옵션 타입 ('BASE' 또는 'ADD')
 * @param int $qty 수량
 * @return int 계산된 단가, 실패시 0
 */
function calculate_aov_price_for_general_mall($it_id, $selected_option, $row_type, $qty) {
    global $g5;
    
    if (!$it_id || !$selected_option || !$row_type || $qty <= 0) {
        return 0;
    }
    
    // AOV 테이블에서 데이터 조회
    $aov_table = isset($g5['g5_shop_item_aov_row_table']) ? 
                 $g5['g5_shop_item_aov_row_table'] : 
                 'gom_mall_item_aov_row';
    
    // QTY row 찾기
    $qty_sql = "SELECT * FROM {$aov_table} 
                WHERE it_id = '".sql_real_escape_string($it_id)."' 
                  AND row_type = 'QTY' 
                LIMIT 1";
    $qty_row = sql_fetch($qty_sql);
    
    if (!$qty_row) {
        return 0;
    }
    
    // 수량 구간에 해당하는 컬럼 인덱스 구하기
    $col_index = get_aov_qty_column_index($qty_row, $qty);
    $col_key = 'col' . $col_index;
    
    // 선택한 옵션 row 찾기
    $option_sql = "SELECT * FROM {$aov_table} 
                   WHERE it_id = '".sql_real_escape_string($it_id)."' 
                     AND row_type = '".sql_real_escape_string($row_type)."' 
                     AND col1 = '".sql_real_escape_string($selected_option)."' 
                   LIMIT 1";
    $option_row = sql_fetch($option_sql);
    
    if (!$option_row) {
        return 0;
    }
    
    // 해당 컬럼의 값 읽기 (단가)
    $unit_price = isset($option_row[$col_key]) ? (int)$option_row[$col_key] : 0;
    
    return $unit_price;
}

/**
 * 수량 구간에 해당하는 컬럼 인덱스 찾기
 * 
 * @param array $qty_row QTY 타입 row
 * @param int $qty 수량
 * @return int 컬럼 인덱스 (2~8), 없으면 2 반환
 */
function get_aov_qty_column_index($qty_row, $qty) {
    $qty = (int)$qty;
    
    if ($qty <= 0) {
        return 2; // 기본값: col2
    }
    
    $col_index = 2; // 기본값: col2
    
    // col2부터 col8까지 순회
    for ($i = 2; $i <= 8; $i++) {
        $col_key = 'col' . $i;
        $col_value = isset($qty_row[$col_key]) ? trim($qty_row[$col_key]) : '';
        
        if ($col_value === '') {
            continue;
        }
        
        // 구간 값이 숫자 이상 형식인지 확인 (예: "1", "10", "50")
        if (preg_match('/^(\d+)\+?$/', $col_value, $matches)) {
            $threshold = (int)$matches[1];
            
            // 다음 컬럼이 있으면 그 이전까지가 구간
            $next_col_key = 'col' . ($i + 1);
            $next_col_value = isset($qty_row[$next_col_key]) ? trim($qty_row[$next_col_key]) : '';
            
            if ($next_col_value !== '') {
                // 다음 구간 시작값 찾기
                if (preg_match('/^(\d+)\+?$/', $next_col_value, $next_matches)) {
                    $next_threshold = (int)$next_matches[1];
                    if ($qty >= $threshold && $qty < $next_threshold) {
                        $col_index = $i;
                        break;
                    }
                }
            } else {
                // 마지막 구간 (이상)
                if ($qty >= $threshold) {
                    $col_index = $i;
                }
            }
        }
    }
    
    return $col_index;
}
```

**검증 방법**:
- 테스트 케이스 작성
- 다양한 수량으로 테스트 (1개, 10개, 50개, 100개 등)
- 폐쇄몰의 `ClosedMallPriceService::calculateAovUnitPrice()`와 결과 비교

```php

    // 필수옵션 추출

    $io_value = $cart_item['ct_option'];

        $after_prefix = trim(substr($io_value, strlen('필수옵션:')));
        $slash_pos = strpos($after_prefix, ' / ');
        if ($slash_pos !== false) {
            $selected_option = trim(substr($after_prefix, 0, $slash_pos));
        } else {
            $selected_option = $after_prefix;
        }
    }
    
    if ($selected_option) {
        // 수량별 가격 재계산
        $new_it_aov_price = calculate_aov_price_for_general_mall(
            $cart_item['it_id'],
            $selected_option,
            'BASE',
            $new_qty
        );
        
        // 장바구니 업데이트
        $update_sql = "UPDATE {$g5['g5_shop_cart_table']}
                      SET ct_qty = '$new_qty',
                          it_aov_price = '$new_it_aov_price',
                          ct_price = '$new_it_aov_price'
                      WHERE ct_id = '$ct_id'";
        sql_query($update_sql);
        
        $unit_price = $new_it_aov_price;
        $total_price = ($new_it_aov_price + $cart_item['io_price']) * $new_qty;

    // 일반 상품: 수량만 업데이트

                   SET ct_qty = '$new_qty'
                   WHERE ct_id = '$ct_id'";
    sql_query($update_sql);
    
    $unit_price = $cart_item['ct_price'];
    $total_price = ($cart_item['ct_price'] + $cart_item['io_price']) * $new_qty;
}

echo json_encode(array(
    'success' => true,
    'unit_price' => $unit_price,
    'total_price' => $total_price,
    'qty' => $new_qty
));
```

## 주의사항

### 1. 폐쇄몰 로직과의 분리
- 일반몰 수정 시 폐쇄몰 로직에 영향을 주지 않도록 주의
- `ct_closed_mall_id IS NULL OR ct_closed_mall_id = 0` 조건 확인
- 폐쇄몰 전용 파일(`closedmall_*.php`)은 수정하지 않음

### 2. 기존 데이터 호환성
- 기존에 저장된 장바구니 데이터는 수정하지 않음
- 새로운 장바구니 추가 시에만 수정된 로직 적용
- 필요시 기존 데이터 마이그레이션 스크립트 작성

### 3. 테스트 환경 구축
- 개발 환경에서 충분한 테스트 후 운영 적용
- AOV 상품과 일반 상품 모두 테스트
- 다양한 수량으로 테스트

### 4. 롤백 계획
- 각 Phase별로 독립적으로 롤백 가능하도록 설계
- Git 브랜치를 활용한 단계별 커밋
- 각 Phase 완료 후 테스트 및 검증
