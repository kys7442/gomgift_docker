# 폐쇄몰 가격 문제 수정 가이드

## 개요

이 문서는 폐쇄몰 전용 가격 관련 문제점들을 수정하기 위한 단계별 실행 가이드입니다.

## 폐쇄몰 가격 정책 개요

### 표시 방침
**시중가(취소선) / 할인받은 금액 / 판매가**

### 가격 표시 타입별 계산 방식

#### 1. 매입가(purchase) 적용 시
- **판매가**: 매입가(it_price)에 할인률을 적용한 금액
- **시중가(취소선)**: 마진율 적용가(it_price2)
- **할인받은 금액**: 시중가 - 판매가

```
매입가(it_price) = 1,000원
할인률 = 10%
→ 판매가 = 1,000 * (1 + 10/100) = 1,100원

마진율 적용가(it_price2) = 1,500원
→ 시중가(취소선) = 1,500원
→ 할인받은 금액 = 1,500 - 1,100 = 400원
```

#### 2. 마진율(margin) 적용 시
- **시중가(취소선)**: 마진율 적용가(it_price2)
- **판매가**: 마진율 금액(it_price2)에서 할인률이 적용된 금액
- **할인받은 금액**: 시중가 - 판매가

```
마진율 적용가(it_price2) = 1,500원
할인률 = 10%
→ 판매가 = 1,500 * (1 - 10/100) = 1,350원
→ 시중가(취소선) = 1,500원
→ 할인받은 금액 = 1,500 - 1,350 = 150원
```

#### 3. 객단가(aov) 적용 시
- **판매가**: 수량별 객단가에서 할인률을 적용한 금액
- **시중가(취소선)**: 객단가 기준가 (마진율 적용가 또는 객단가 기준가)
- **할인받은 금액**: 시중가 - 판매가 (수량별로 다름)

```
수량 1개: 객단가 = 1,000원, 할인률 10% → 판매가 = 900원
수량 10개: 객단가 = 800원, 할인률 10% → 판매가 = 720원
수량 50개: 객단가 = 600원, 할인률 10% → 판매가 = 540원
```

### 중요 사항

1. **각 폐쇄몰마다 가격 정책이 다름**
   - `cm_price_display_type`: purchase, margin, aov
   - `cm_discount_rate`: 할인/할증율 (%)
   - `cm_price_round_unit`: 올림 단위 (10, 100, 1000)

2. **일관성 유지 필요**
   - 상품 리스트
   - 상품 상세 페이지
   - 옵션 선택 시 스크립트 계산
   - 장바구니 담기 시 가격
   - 주문 정보
   - 결제 완료

3. **일반몰과 완전 분리**
   - 폐쇄몰 전용 파일 사용: `closedmall_*.php`
   - 폐쇄몰 전용 JavaScript: `shop.closed.js`
   - `ct_closed_mall_id`로 구분

## 폐쇄몰 가격 관련 문제점 요약

### 1. 가격 계산 함수 중복 및 불일치
- `get_closed_mall_price()` (closed_mall.lib.php)
- `get_closed_mall_price_display()` (closed_mall_price_display.lib.php)
- `ClosedMallPriceService::getUnitPrice()` (closed_mall_price_service.php)
- **문제**: 어떤 함수를 언제 사용해야 하는지 명확하지 않음

### 2. PHP와 JavaScript 계산 로직 불일치
- PHP: `floor()` 사용, 올림 처리 방식
- JavaScript: `Math.floor()`, `Math.ceil()` 사용 방식 불일치
- **문제**: 서버와 클라이언트 계산 결과가 다를 수 있음

### 3. 가격 표시 형식 불일치
- 상품 리스트: 일부만 표시 형식 적용
- 상품 상세: 다른 표시 형식 사용
- 장바구니: 또 다른 표시 형식 사용
- **문제**: 동일 상품이 다른 가격으로 표시될 수 있음

### 4. AOV 상품 수량별 가격 재계산 문제
- 상품 상세 페이지: 수량 변경 시 가격 재계산
- 장바구니: 수량 변경 시 가격 재계산
- **문제**: 수량별 가격이 일치하지 않을 수 있음

### 5. 할인 금액 계산 오류
- 매입가 기준: 할인 금액 계산 방식 불일치
- 마진율 기준: 할인 금액 계산 방식 불일치
- 객단가 기준: 수량별 할인 금액 계산 누락
- **문제**: 할인받은 금액이 잘못 표시될 수 있음

#### 작업 내용

**1-1. 통합 가격 계산 함수 생성**

파일: `lib/closed_mall_price_unified.php` (신규 생성)

```php
<?php
if (!defined('_GNUBOARD_')) exit;

/**
 * 폐쇄몰 통합 가격 계산 함수
 * 
 * 모든 폐쇄몰 페이지에서 동일한 가격 계산 로직을 사용하도록 통합
 * 
 * @param array $item 상품 정보
 * @param array $cm 폐쇄몰 정보
 * @param int $qty 수량 (AOV 상품인 경우)
 * @param string $selected_option 선택한 옵션 (AOV 상품인 경우)
 * @param bool $is_member 로그인 여부
 * @return array 가격 정보
 */
function get_closed_mall_price_unified($item, $cm, $qty = 1, $selected_option = '', $is_member = null) {
    global $is_member;
    
    if ($is_member === null) {
        $is_member = isset($is_member) ? $is_member : false;
    }
    
    $price_display_type = isset($cm['cm_price_display_type']) ? $cm['cm_price_display_type'] : 'purchase';
    $discount_rate = isset($cm['cm_discount_rate']) ? floatval($cm['cm_discount_rate']) : 0;
    $price_round_unit = isset($cm['cm_price_round_unit']) ? intval($cm['cm_price_round_unit']) : 0;
    
    // AOV 상품인 경우 ClosedMallPriceService 사용
    if (isset($item['it_aov_use']) && $item['it_aov_use'] == 'Y' && $price_display_type == 'aov') {
        include_once(G5_LIB_PATH.'/closed_mall_price_service.php');
        
        $aov_data = ClosedMallPriceService::loadAovData($item['it_id']);
        $unit_price = ClosedMallPriceService::getUnitPrice(
            $item,
            $selected_option,
            'BASE',
            $qty,
            $price_display_type,
            $discount_rate,
            $aov_data,
            $price_round_unit
        );
        
        // 시중가(취소선) = 마진율 적용가
        $cancel_price = isset($item['it_price2']) ? (int)$item['it_price2'] : 0;
        
        // 할인 금액 계산
        $discount_amount = 0;
        if ($cancel_price > 0 && $cancel_price > $unit_price) {
            $discount_amount = $cancel_price - $unit_price;
        }
        
        return array(
            'display_price' => $unit_price,
            'cancel_price' => $cancel_price,
            'discount_amount' => $discount_amount,
            'discount_rate' => $discount_rate,
            'price_display_type' => $price_display_type,
            'unit_price' => $unit_price
        );
    }
    
    // 일반 상품인 경우 get_closed_mall_price_display 사용
    include_once(G5_LIB_PATH.'/closed_mall_price_display.lib.php');
    return get_closed_mall_price_display($item, $cm, $is_member, $qty, $selected_option);
}
```

```php
/**
 * 폐쇄몰 가격 표시 HTML 생성
 * 
 * 표시 형식: 시중가(취소선) / 할인받은 금액 / 판매가
 * 
 * @param array $price_info get_closed_mall_price_unified() 결과
 * @param array $options 표시 옵션
 * @return string HTML
 */
function format_closed_mall_price_display_unified($price_info, $options = array()) {
    $display_price = isset($price_info['display_price']) ? (int)$price_info['display_price'] : 0;
    $cancel_price = isset($price_info['cancel_price']) ? (int)$price_info['cancel_price'] : 0;
    $discount_amount = isset($price_info['discount_amount']) ? (int)$price_info['discount_amount'] : 0;
    
    $html = '';
    
    // 시중가(취소선) 표시
    if ($cancel_price > 0 && $cancel_price > $display_price) {
        $html .= '<span class="cm_cancel_price" style="text-decoration: line-through; color: #999; margin-right: 8px;">';
        $html .= number_format($cancel_price) . '원</span>';
    }
    
    // 할인받은 금액 표시
    if ($discount_amount > 0) {
        $html .= '<span class="cm_discount_amount" style="color: #FF8C42; margin-right: 8px;">';
        $html .= '할인 ' . number_format($discount_amount) . '원</span>';
    }
    
    // 판매가 표시
    $html .= '<span class="cm_display_price" style="color: #333; font-weight: bold;">';
    $html .= number_format($display_price) . '원</span>';
    
    return $html;
}
```

#### 작업 내용

**3-1. `js/shop.closed.js` 수정**

**현재 문제**:
- `calculate_closed_mall_price()` 함수의 계산 로직이 PHP와 다름
- `floor()` 사용 여부 불일치
- 올림 처리 방식 불일치

**수정 후 코드**:
```javascript
function calculate_closed_mall_price(item, price_display_type, discount_rate, price_round_unit, is_member, qty) {
    var it_price = parseInt(item.it_price) || 0;
    var it_price2 = parseInt(item.it_price2) || 0;
    var margin_price = it_price2;
    
    var display_price = 0;
    var cancel_price = 0;
    var discount_amount = 0;
    
    switch(price_display_type) {
        case 'purchase':
            // 매입가에 할인률 적용
            if (it_price > 0) {
                if (discount_rate > 0) {
                    // 할증: 매입가 * (1 + 할인률/100)
                    display_price = Math.floor(it_price * (1 + (discount_rate / 100)));
                } else if (discount_rate < 0) {
                    // 할인: 매입가 * (1 - |할인률|/100)
                    display_price = Math.floor(it_price * (1 - (Math.abs(discount_rate) / 100)));
                } else {
                    display_price = it_price;
                }
                
                // 원단위 올림 처리
                if (price_round_unit > 0 && [10, 100, 1000].indexOf(price_round_unit) !== -1) {
                    display_price = Math.floor(Math.ceil(display_price / price_round_unit) * price_round_unit);
                }
            }
            
            // 시중가(취소선) = 마진율 적용가
            cancel_price = margin_price;
            
            // 할인 금액 = 시중가 - 판매가
            if (cancel_price > 0 && cancel_price > display_price) {
                discount_amount = cancel_price - display_price;
            }
            break;
            
        case 'margin':
            // 시중가(취소선) = 마진율 적용가
            cancel_price = margin_price;
            
            // 판매가 = 마진율 금액에서 할인률 적용
            if (margin_price > 0) {
                if (discount_rate > 0) {
                    discount_amount = Math.floor(margin_price * (discount_rate / 100));
                    display_price = margin_price - discount_amount;
                } else {
                    display_price = margin_price;
                    discount_amount = 0;
                }
                
                // 원단위 올림 처리
                if (price_round_unit > 0 && [10, 100, 1000].indexOf(price_round_unit) !== -1) {
                    display_price = Math.floor(Math.ceil(display_price / price_round_unit) * price_round_unit);
                    discount_amount = cancel_price - display_price;
                }
            }
            break;
            
        case 'aov':
            // AOV는 서버에서 계산 (ajax_closedmall_price.php)
            // 클라이언트에서는 기본값만 계산
            cancel_price = margin_price;
            display_price = margin_price;
            discount_amount = 0;
            break;
    }
    
    return {
        display_price: display_price,
        cancel_price: cancel_price,
        discount_amount: discount_amount
    };
}
```

## 주의사항

### 1. 일반몰과의 완전 분리
- 폐쇄몰 수정 시 일반몰 로직에 영향을 주지 않도록 주의
- `ct_closed_mall_id > 0` 조건 확인
- 일반몰 전용 파일(`shop/*.php`)은 수정하지 않음

### 2. 각 폐쇄몰별 가격 정책
- 각 폐쇄몰마다 `cm_price_display_type`, `cm_discount_rate`, `cm_price_round_unit`이 다름
- 폐쇄몰 정보를 올바르게 조회하여 사용해야 함

### 3. 로그인/비로그인 구분
- 비로그인 시에도 폐쇄몰 가격 정책 적용 (요구사항에 따라 다를 수 있음)
- 로그인 여부에 따라 가격 표시 방식이 달라질 수 있음

### 4. 테스트 환경 구축
- 개발 환경에서 충분한 테스트 후 운영 적용
- 각 가격 표시 타입(purchase, margin, aov) 모두 테스트
- 다양한 수량으로 테스트

### 5. 롤백 계획
- 각 Phase별로 독립적으로 롤백 가능하도록 설계
- Git 브랜치를 활용한 단계별 커밋
- 각 Phase 완료 후 테스트 및 검증
