## 개발 환경 설정

### 1. 환경 변수 설정
`.env.local` 파일을 생성하고 다음 내용을 입력하세요:

```env
DB_HOST=115.68.222.162
DB_USER=gongsa
DB_PASSWORD=@Dudtlr00_2025!@
DB_NAME=idb07832
DB_PREFIX=g5_

SESSION_SECRET=your-secret-key-change-this
JWT_SECRET=your-jwt-secret-change-this

NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME=모두시공
```

### 2. 의존성 설치
```bash
cd intx-nextjs
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

서버가 실행되면 `http://localhost:3000`에서 접속할 수 있습니다.

## 알려진 이슈 및 제한사항

1. **파일 업로드 경로**: 현재 `/public/data/file/`에 저장됩니다. 실제 서버에서는 기존 경로와 동기화가 필요할 수 있습니다.

2. **댓글 새로고침**: 댓글 작성 후 자동 새로고침이 필요합니다.

3. **공지사항**: 공지사항 처리 로직이 간단하게 구현되어 있습니다.

4. **검색 기능**: 일부 검색 옵션이 아직 완전히 구현되지 않았을 수 있습니다.
