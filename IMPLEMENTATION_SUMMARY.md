## ⚠️ 주의사항

1. **파일 업로드 경로**: `/public/data/file/` 폴더에 저장됩니다. 실제 서버에서는 기존 `/data/file/` 경로와 동기화가 필요할 수 있습니다.

2. **세션 관리**: 현재 쿠키 기반 세션을 사용합니다. 실제 배포 시 보안 설정을 확인하세요.

3. **데이터베이스**: 기존 그누보드5 데이터베이스를 그대로 사용합니다. 테이블 구조 변경 없이 호환됩니다.

4. **URL 구조**: 
   - 기존: `/bbs/board.php?bo_table=free`
   - 신규: `/bbs/board/free`
   - 기존 URL과의 호환성을 위해 리다이렉트 설정이 필요할 수 있습니다.
